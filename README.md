
# conv

An util to convert text-based format.

- TSV <-> json
- FS files <-> rgrepn <-> json

## supported format

- See [FORMAT](./FORMAT.md)

## dependencies

Mainly:

- posix/shell (or bash)
- [jq](https://stedolan.github.io/jq/)
- grep
- base64

Optionnal:
- mlr : [miller](https://miller.readthedocs.io/)

## hex support
- od, tr (from coreutils)
- lua

## xml support

- tinyxml


## yaml supprt

- python
- [yaml2json]()
